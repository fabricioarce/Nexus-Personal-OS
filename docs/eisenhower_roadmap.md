# Eisenhower Module - ROADMAP

## Phase 1: MVP (Current Focus)
- [x] Basic CRUD for tasks.
- [x] Scatter plot visualization (Importance vs Urgency).
- [x] Fast entry form.
- [x] Basic notes support.

## Phase 2: Refinement
- [ ] **Advanced Filtering**: Filter by category or time range.
- [ ] **Drag & Drop**: Reposition tasks on the matrix to update their ratings visually.
- [ ] **Rich Text Notes**: Support for markdown in task notes.

## Phase 3: Integrations
- [ ] **Google Calendar Sync**:
  - Automatically import events as tasks.
  - Export urgent/important tasks as calendar events.
- [ ] **Notifications**: Alerts for tasks needing immediate attention.

## Phase 4: Intelligence
- [ ] **AI Prioritization**: Suggest ratings based on task description.
- [ ] **Performance Analytics**: Weekly report on how many "Important/Not Urgent" tasks were completed vs "Urgent/Not Important".
